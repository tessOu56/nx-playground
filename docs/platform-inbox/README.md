# Platform inbox (receiving zone)

Tickets and roadmap excerpts are **distributed from** [platform-command](https://github.com/tessOu56/platform-command).

| File | Purpose |
|------|---------|
| `CURRENT.md` | Active P0/P1 list — **start here** |
| `tickets/T-*.md` | One file per ticket |
| `MANIFEST.json` | Machine manifest (ids, sync time) |

After completing work, set `status: done` in the ticket file footer or open a GitHub Issue and link `issueUrl`.
